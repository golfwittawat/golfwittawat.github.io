หน้าแบบที่ 1 เน้นภาพ “ระบบจริง” และ flow การใช้งานครบทั้งระบบ

เวอร์ชันนี้สร้างจากโครง route และฟีเจอร์จริงของโปรเจกต์ projects/promayan/promayan_card โดยย่อให้เป็น mockup ที่อ่านง่าย เห็นชัดทั้งฝั่งผู้ใช้และฝั่งแอดมิน ทำให้เห็นได้ทันทีว่าแอพไม่ได้มีแค่หน้าหลัก แต่มีระบบครบตั้งแต่ login/register, รีวิวไพ่, ดูดวง, ประวัติ, profile, แก้ไขเนื้อหาเสียง ไปจนถึงตั้งค่าระบบและ route staff-only

Route จริง: Login / Register / Review / Divine / History / Profile / Settings / Admin / CardEditor Vue 3 + Django REST + PostgreSQL รองรับ TTS, ElevenLabs, Upload Audio ธีมจริง: Cosmic Dark + Gold Accent + Sacred Visual
เริ่มจากหน้าทบทวนไพ่ ดูฝั่งแอดมิน
9 route หลักใน frontend รวม public, member และ staff
3 รูปแบบการดูดวง 1 ใบ, 3 ใบ, 12 ภพ
2 ระดับสิทธิ์หลัก Member และ Admin พร้อม route guard
จุดขายของแบบนี้ เห็นภาพชัดว่าระบบมีทั้งหน้าบ้านและหลังบ้านครบ เหมาะกับการดูความพร้อมใช้งานจริง ความต่อเนื่องของ flow และการต่อยอดในระยะยาว

ภาพรวมหน้าหลักของระบบ

ด้านล่างคือ mockup ของหน้าสำคัญที่ดึงมาจากระบบต้นทาง ทั้ง flow เข้าสู่ระบบ ผู้ใช้ทั่วไป และแอดมิน จัดเลย์เอาต์ใหม่ให้อ่านง่ายขึ้น แต่ยังเก็บสาระของระบบจริงไว้ครบ
