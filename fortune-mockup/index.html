พรหมญาณ App Overview

ภาพรวมแนวทางหน้าจอและรูปแบบการใช้งานของระบบพรหมญาณ

ดูทั้ง 3 แบบ
App Variations จากโปรเจกต์ promayan_card

สำรวจ 3 แนวทางของหน้าจอระบบพรหมญาณในมุมมองที่ต่างกัน

หน้านี้ออกแบบมาเพื่อรวมเว็บไซต์พรหมญาณทั้ง 3 รูปแบบไว้ในหน้าเดียว โดยแบบที่ 1 จะอ้างอิง route และสิทธิ์ผู้ใช้จากโปรเจกต์ promayan_card โดยตรง ส่วนแบบที่ 2 และ 3 จะช่วยให้เห็นมุมมองเชิงฟังก์ชันและเชิงภาพลักษณ์ของแอพ เพื่อใช้เปรียบเทียบจุดเด่นของแต่ละแนวทางได้ชัดเจนขึ้น

เริ่มดูตัวเลือกทั้งหมด เปิดแบบที่ 1 ทันที
แบบที่ 1

อิงจากระบบจริงของโปรเจกต์ต้นทาง ครอบคลุม login, review, divine, history, profile, settings และฝั่ง admin

แบบที่ 2

เน้น flow การใช้งานและการจัดการระบบ เหมาะกับการดูภาพรวมการทำงานแบบ step-by-step

แบบที่ 3

เน้นบรรยากาศและภาพลักษณ์ของแอพ เหมาะกับการสื่อ mood, tone และความรู้สึกของการใช้งาน

เปรียบเทียบแนวทางของแอพได้ในหน้าเดียว มีหน้ากลางสำหรับเปรียบเทียบทั้ง 3 แบบในที่เดียว ช่วยให้เห็นความต่างของ route จริง ฟังก์ชัน ระบบหลังบ้าน และโทนภาพรวมของแอพได้ชัดขึ้น

รูปแบบเว็บไซต์ทั้ง 3 แบบ

เลือกเปิดดูแต่ละแบบได้ทันทีจากปุ่มด้านล่าง ถ้าจะใช้หน้านี้ขึ้น GitHub Pages หน้านี้ก็จะเป็นหน้าหลักสำหรับเข้าไปดูแต่ละเวอร์ชันของระบบ โดยแบบที่ 1 จะเหมาะที่สุดเมื่ออยากดูโครงที่ผูกกับระบบจริงมากที่สุด

แบบที่ 1 System Product Mockup

ภาพระบบจากโปรเจกต์จริง

เวอร์ชันนี้นำ route, สิทธิ์ผู้ใช้, และ flow หลักจากโปรเจกต์ promayan_card มาทำเป็น mockup ของระบบใช้งานจริง จึงเหมาะกับการดูโครงสร้างหน้าจอ และความเชื่อมโยงของฟังก์ชันหลักมากกว่าแค่หน้า concept

  • มี mockup ของ route หลักทั้งฝั่งสมาชิกและ route เฉพาะ staff
  • สะท้อนโครงจริงของ review, divine, history, profile, settings, admin และ card editor
  • แสดงการจัดการไพ่ เนื้อหา และระบบเสียงแบบ Edge TTS, ElevenLabs, Upload ได้ชัดเจน

แบบที่ 2 Mockup พร้อมใช้งาน

หน้าระบบและการจัดการ

เวอร์ชันนี้เน้นโครงหน้าจัดการ ชุดไพ่ และ flow การใช้งานที่ต่อเนื่อง เหมาะกับการดูภาพรวมเชิงฟังก์ชันของระบบ

  • เห็นภาพการใช้งานจริงและหน้าจัดการชัด
  • มีภาพไพ่จริงแทรกในหลายจุดของ mockup
  • เหมาะกับการดูความครบของระบบและลำดับขั้นตอนการใช้งาน

แบบที่ 3 Brand Experience Style

งานขายภาพลักษณ์

เวอร์ชันนี้เน้นอารมณ์และภาพลักษณ์ของพรหมญาณ เหมาะกับการสื่อความรู้สึก ความพรีเมียม และบรรยากาศของแอพ

  • เหมาะกับการสื่อ mood และ tone ของหน้าแรกหรือ landing page
  • เน้นความรู้สึกและความน่าเชื่อถือของแบรนด์
  • ใช้เป็นตัวเลือกสำหรับแนวทางหน้าเว็บไซต์หลัก
